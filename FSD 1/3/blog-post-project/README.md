# Blog Post Project

This project is a simple responsive web page that displays a single blog post. It includes the following components:

- **Title**: The main title of the blog post.
- **Author Information**: Details about the author of the post.
- **Post Content**: The main content of the blog post.
- **Comment Section**: A section for users to submit comments, including a form for input.

## Project Structure

The project is organized as follows:

```
blog-post-project
├── index.html          # Main HTML document for the blog post layout
├── css
│   └── styles.css     # External CSS styles for the blog post layout
├── js
│   └── main.js        # JavaScript code for interactive elements
├── vendor
│   └── bootstrap
│       ├── css
│       │   └── bootstrap.min.css  # Minified CSS version of Bootstrap
│       └── js
│           └── bootstrap.bundle.min.js  # Minified JS version of Bootstrap
├── .gitignore          # Specifies files to be ignored by Git
├── package.json        # Configuration file for npm
└── README.md           # Documentation for the project
```

## Getting Started

To set up the project locally, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd blog-post-project
   ```

3. **Open `index.html` in your web browser** to view the blog post layout.

## Dependencies

This project uses Bootstrap for responsive design and styling. The Bootstrap files are included in the `vendor/bootstrap` directory.
